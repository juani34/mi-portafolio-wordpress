# mi-portafolio-wordpress
# E-commerce con API de Clima  

## Tecnologías usadas  
- WordPress + GeneratePress  
- PHP (plugin custom)  
- WooCommerce  

## Cómo probarlo localmente  
1. Clona este repositorio.  
2. Copia `plugin-clima.php` a `/wp-content/plugins/`.  
3. Activa el plugin en WordPress.
